# Entri Jurnal 3: Teknik Dasar Analisis Statis
**Tanggal:** 6 Juli 2026

* **Apa yang dipahami:** Hari ini saya mempelajari teknik analisis statis tanpa menjalankan program. Saya menggunakan `strings` untuk melihat pesan-pesan teks di dalam *binary* dan melihat *Import Address Table (IAT)* untuk mengetahui fungsi sistem apa saja yang dipanggil oleh program tersebut.
* **Apa yang masih bingung:** Saya masih sering bingung membedakan fungsi API yang umum digunakan oleh pengembang dengan API yang sering disalahgunakan oleh malware.
* **Refleksi:** Analisis statis memberikan gambaran besar perilaku program sebelum kita melakukan *debugging* yang lebih mendalam.
